## Decision rules
- Prioritize queries that can drive signups and activated users, not vanity traffic.
- Prefer improving an existing page before creating a near-duplicate page.
- Only create a new page when intent is distinct.
- Tie every recommendation back to Noeis’s category wedge: reliable recall, concept formation, human-centered AI, and serious reading workflows.
